清空提交记录 删除所有提交记录



可以，最直接的方法就是：**重新创建一个只有当前代码的新仓库历史，然后强推到远程**。

### 1）进入项目目录
```bash
cd /path/to/your/project
```

### 2）删除旧的 Git 历史
```bash
rm -rf .git
```

这一步会删除本地所有提交记录、分支信息、tag 等 Git 元数据，但**不会删除你的项目文件**。

---

### 3）重新初始化仓库
```bash
git init
git config user.name xxx
git config user.email xxx@xxx.xxx
git branch -m main
echo test>test.txt
git add test.txt
git commit
git checkout -b xxxxxxxx
```

### 4）重新添加远程仓库
把下面的地址换成你的 GitHub 仓库地址：
```bash
git remote add origin https://github.com/yourname/yourrepo.git
```

---

### 5）重新提交当前代码
```bash
git add .
git commit -m "Initial commit"
```

---

### 6）强制推送覆盖远程

```bash
git push -f origin --all
```
