1. 安装

   pip install git-filter-repo

2. 修改

   执行

   ```
   git filter-repo --commit-callback '
   if commit.author_name == b"wrongname":
       commit.author_name = b"david"
       commit.author_email = b"david@david.com"
   if commit.committer_name == b"wrongname":
       commit.committer_name = b"david"
       commit.committer_email = b"david@david.com"
   ' --force

3. 强制提交，hash已改变，需要设置远程仓库地址、分支

4. 其他设备直接重新clone
