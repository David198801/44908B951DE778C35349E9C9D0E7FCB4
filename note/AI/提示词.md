# 漫画爬虫

写一个python调用chrome driver的漫画爬虫
1. 从“爱奇艺叭嗒.txt”里获取作品url
2. 打开作品url，获取第一个拥有data-comicid、data-comictitle的标签，得到漫画id和漫画标题，以漫画标题创建目录
3. 调用“https://www.iqiyi.com/manhua/catalog/{漫画id}/”接口，获取到漫画目录json，访问json的episodes得到章节数组，数组的episodeTitle为章节名，episodeId为章节id，在漫画标题目录内创建章节名目录
4. 访问“https://www.iqiyi.com/manhua/reader/{漫画id}_{章节id}.html”进入章节，获取<ul class="main-container">里所有的img（间接包含），获取img的data-original属性，没有则获取src属性，得到图片url
5. 下载图片，以下载顺序命名为001、002格式
6. 支持输出日志，支持将获取到的章节信息缓存到json中，支持记录下载完成的章节，支持下载失败则重试，支持每章节下载完成后暂停指定秒数
