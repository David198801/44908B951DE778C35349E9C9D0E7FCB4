# 导出凭据（无法查看密码）

1. win+r执行rundll32.exe keymgr.dll,KRShowKeyMgr打开凭据导出功能
2. 选择，导出并设置密码为1
3. 另一台电脑导入
或直接用凭据管理器的备份、导入功能
# mimikatz查看
1. 下载 https://github.com/gentilkiwi/mimikatz/releases
2. CMD管理员运行，cd进入mimikatz的x64目录，运行mimikatz
3. 执行`log`记录日志
4. 执行`privilege::debug`获取调试权限（必须）
5. 执行`sekurlsa::logonpasswords`列出当前登录用户凭据
